Sequences

The Sequence File Format

Generators

  • randomSeq: Generates sequences of values uniformly at random in a given range.
  • exptSeq: Generates sequences of values with repeats appearing in an exponential distribution.
  • almostSortedSeq: Generates almost sorted sequences.
  • trigramSeq: Generates sequences of character strings generated using at random using a trigram probability distribution.
  • addDataSeq: Adds auxiliary data to a sequence of a simple type.
